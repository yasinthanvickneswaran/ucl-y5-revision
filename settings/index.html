Settings — Sync

Connect a GitHub Gist to keep your timetable progress, quiz scores and weakness flags in sync across all your devices.

✓ Gist sync is connected

1

Create a GitHub personal access token

Go to github.com/settings/tokens/new (classic token). Give it a name like "UCL revision sync" and tick only the gist scope. Set expiry to "No expiration" or 1 year. Copy the token — you only see it once.

2

Enter your token

Stored only in your browser. Never sent anywhere except the GitHub API.

3

Done — sync on every device

Open Settings on any other device and repeat steps 1–2 with the same token. Your progress will pull automatically when each page loads.

Use existing Gist (second device)

If you've already set this up on another device, enter your existing Gist ID to connect to the same data.
